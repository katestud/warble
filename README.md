# Warble

This follows the [LiveView Twitter clone demo](https://www.phoenixframework.org/blog/build-a-real-time-twitter-clone-in-15-minutes-with-live-view-and-phoenix-1-5) presented by Chris McCord.

LiveView is integrated with:
* Post create
* Post update
* Post like
* Post repost

:rocket: :rocket: :rocket:

To start your Phoenix server:

  * Setup the project with `mix setup`
  * `mix ecto.create` && `mix ecto.migrate`
  * Start Phoenix endpoint with `mix phx.server`

Now you can visit [`localhost:4000`](http://localhost:4000) from your browser.
